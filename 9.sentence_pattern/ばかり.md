# ばかり
## 表示名词的限定

 #std_l36  

```nihongo
[n.] ばかり  
```

可译为: 都是  
表示列举的事物全部相同  

## 表示动词的限定  

 #std_l36  

```nihongo
[v.conj.] て ばかりいます  
```

可译为: 一直，总是  
表示动作的重复或持续，带有负面意义  

## 表示动作或事情刚刚结束

 #std_l40  

```nihongo
[v.conj.] た ばかりです  
```

表示动作刚刚结束，类似 [`~たところ`](ところ.md#表示动作刚刚结束)  

相较于 [`~たところ`](ところ.md#表示动作刚刚结束)，`~ばかり` 可以表示动作行为结束了很长一段时间之后的事态
