# ても/でも

## でも作连词表示逆接/转折
 #l10
 
```nihongo
[sentence.]。でも，[sentence.]
```

表示转折关系，用于口语  
## 作助词  
 #l15
```nihongo
[n.]/[adj.na.conj.]でも [clause.]
[adj.i.conj.]/[v.conj.II.]ても [clause.]
```

可译为: 即使，无论，就算  
表示假定条件的逆接  

特别地，该句式也可衍生用于表示允许或禁止  
原意为"即使~也可以/不行"，衍生用于表示允许或禁止

```nihongo
~ても/でも いいです/かまわない
~ても/でも いけません
```
