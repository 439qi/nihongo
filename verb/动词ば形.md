# 动词ば形  

常接续 `ば` 构成动词ば形用于表示假定条件  

| 分类   | 规则                                                                     |
| ---- | ---------------------------------------------------------------------- |
| 一类动词 | う段 => え段+ば                                                             |
| 二类动词 | る => れ+ば                                                               |
| 三类动词 | する => すれ+ば<br><ruby>来<rt>く</rt></ruby>る => <ruby>来<rt>く</rt></ruby>れ+ば |
