|                            |              |
| -------------------------- | ------------ |
| きる / 着る / kiru             | 穿（上身或全身）     |
| はく / 履く / haku             | 穿（下身）        |
| かぶる / 被る / kaburu          | 戴（头部）        |
| つける / 着ける / tsukeru **する** | 戴；系上；佩带（小物件） |


当你“穿上”衣服，你就是穿着它们。日语中没有特别区分这样的用词。当我们讨论“穿着”某物时，会用 **て形** 和 **いる**.

|   |   |   |
|---|---|---|
|还没有|コートを着る。（我（会）穿上外套。）|コートを着ない。（我不（准备）穿外套。）|
|已经|コートを着ている。（我穿着外套。）|コートを着ていない。（我没有穿外套。）|
|已经（更加随意）|コートを着てる。（我穿着外套。）|コートを着てない。（我没有穿外套。）|
