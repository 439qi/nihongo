# 动词ない形  

 #📖ref/std/19  
原形 => ない形  

|分类|规则|
|-| - |
|一类动词|非う结尾 => あ段+ない<br>う结尾 => わ+ない<br>==特例: ある => ない==|
|二类动词|る => ない|
|三类动词|する => しない<br><ruby>来<rt>く</rt></ruby>る => <ruby>来<rt>こ</rt></ruby>ない|
