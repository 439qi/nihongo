由动词未然形接续 **よう** 构成  
敬体意志形由 [**ます**](../5.auxi_verb/ます.md) 的未然形接续 **よう** 构成，即 [ましょう](../9.sentence_pattern/ましょうか、ませんか.md)  
用于表示说话人的决心、意志的表达形式，也可用于表示提议  

| 分类   | 规则                                                               |
| ---- | ---------------------------------------------------------------- |
| 一类动词 | う段->お段长音                                                         |
| 二类动词 | る->よう                                                            |
| 三类动词 | する->しよう<br><ruby>来<rt>く</rt></ruby>る-><ruby>来<rt>こ</rt></ruby>よう |
