# か  

## 终助词表示敬体疑问句  

```nihongo
[sentence.]か  
```

## 副助词表示对若干项进行选择  

 #l7  

可译为 "或者"  
最后一个 `か` 可省略  

```nihongo
[n.]か[n.]か  
[adj.]か[adj.]か  
[v.basic.]か[v.basic.]か  
```

## 接续在疑问词后表某个  

`[pron.interro.]か`  
表示该主体是不明确的，或者不需要说明  
其后助词为 `を` `が` 时省略助词  

## 接在小句后表示不确定的内容  

 #l23  

```nihongo
[clause.final.] か [v.]  
```

当小句不带疑问代词时，其后接[かどうか](かどうか.md)  
当小句带有 <code><ruby>何<rt>なに</rt></ruby></code>、誰 等疑问代词时，直接使用该句式
