# ます

**ます** 用于动词敬体，称为敬体助动词  

| 活用形 | 词尾变形 |
| --- | ---- |
| 未然形 | ましょ  |
|     | ませ   |
| 连用形 | まし   |
| 终止形 | ます   |
| 连体形 | ます   |
| 假定形 | ますれ  |
| 命令形 | ませ   |
|     | まし   |

**ます** 连体形只用于接续 **の**、[**ので**](../4.particle/ので.md)、**のに**  

**ます** 假定形及命令形通常不会使用，动词表示假定时通过活用变形

| |现在|过去|
|-|-|-|
|肯定|~ます|~ました|
|否定|~ません|ませんでした|
