---
title: ぬ
---
**ぬ** 也是 [**ない**](ない.md) 的古语，同时也是 [**ず**](ず.md) 的连体形  
特别地，对于**する** 使用 **せぬ**，对于 **くる** 使用 **こぬ**  

| 活用形 | 词尾变形 |
| --- | ---- |
| 未然形 | な    |
| 连用形 | に    |
| 终止形 | ぬ    |
| 连体形 | ぬる   |
| 已然形 | ぬれ   |
| 命令形 | ね    |

> [!example] 现代多用在诗歌、谚语或很严谨的文章  
> 年には勝てぬ。
> > 岁月不饶人。
> 
> 言わぬが花。
> > 不说为妙。
