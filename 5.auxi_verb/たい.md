# たい

 #std_l17  

> [!note] Grammar
> [v.conj.] たい

希望助动词 **たい** 表示希望进行某个动词的动作  
特别地，不能前接 **ある**  

| 活用形 | 词尾变形 | 常接助词/助动词 |
| --- | ---- | -------- |
| 未然形 | たかろ  | う        |
| 连用形 | たかっ  | た        |
|     | たく   | て、ない     |
| 终止形 | たい   |          |
| 连体形 | たい   |          |
| 假定形 | たけれ  |          |

**たい** 通常只用于描述自身的愿望，对于第三人称使用 [**がる**](../4.particle/がる.md) 动词化构成 **たがる**  
当句子为疑问句、[引用内容](../4.particle/と.md#表示引用内容)、[连体修饰语or定语](../9.sentence_pattern/连体修饰语or定语.md)时，无需使用 **がる**
