 #moc

<font color=red>大部分</font>情况下用于接续动词

## たい
 #l17
`たい` 表示希望进行某个动词的动作，称为希望助动词  

| 活用形 | 词尾变形 |
| --- | ---- |
| 未然形 | たかろ  |
| 连用形 | たかっ  |
|     | たく   |
| 终止形 | たい   |
| 连体形 | たい   |
| 假定形 | たけれ  |
