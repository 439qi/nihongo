# です

> `です` 的敬语为 `でございます`

|活用形|词尾变形|
|-|-|
|未然形|でしょ|
|连用形|でし|
|终止形|です|
|连体形|です|

`です` 连体形只用于接续 [`ので`](../4.particle/ので.md)、`のに`  

下表中的 `では` 在口语中可替换为 `じゃ`  

| 时态    | 肯否  | 敬体                      | 简体     |
| ----- | --- | ----------------------- | ------ |
| 现在与将来 | 肯定  | です                      | だ      |
|       | 否定  | ではありません<br>/ではないです      | ではない   |
| 过去    | 肯定  | でした                     | だった    |
|       | 否定  | ではありませんでした<br>/ではなかったです | ではなっかた |
