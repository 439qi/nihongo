**ず** 是 [**ない**](ない.md) 的古语  
特别地，接续 **ず** 时，**する** 未然形为 **せ** 而非 **し**  

## 活用形

| 活用形 | 词尾变形 |
| --- | ---- |
| 未然形 |      |
| 连用形 |      |
| 终止形 | ず    |
| 连体形 |      |
| 已然形 |      |
| 命令形 |      |

> 另有 **ずに**
> **ずに** 前文不可独立，若独立通常为省略用法
> **ずに** 对应于 [**ないで**](ない.md#ないで%20与%20なくて)
> **ず** 对应于 [**なくて**](ない.md#ないで%20与%20なくて)
