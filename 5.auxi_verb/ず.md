**ず** 是 [**ない**](ない.md) 的古语  
特别地，对于**する** 使用 **せず**，对于 **くる** 使用 **こず**  
## 活用形  

| 活用形 | 词尾变形 |
| --- | ---- |
| 未然形 | (ず)  |
| 连用形 | ず    |
| 终止形 | ず    |
| 连体形 | ぬ    |
| 已然形 | ね    |
| 命令形 | ざれ   |

另有 **ずに**  
**ずに** 前文不可独立，若独立通常为省略用法  
**ずに** 对应于 [**ないで**](ない.md#ないで%20与%20なくて)  
**ず** 对应于 [**なくて**](ない.md#ないで%20与%20なくて)
